## Introduction

This cheatsheet is a collection of hotkeys for Dwarf Fortress, describing the default hotkeys for version v53.03.

## How to read

The keyboard layout is based on the QWERTY keyboard layout and shows the default hotkeys for version Dwarf Fortress.  

* Blue commands :large_blue_circle:: main hotkeys
* Green commands :green_circle:: main hotkeys needing SHIFT
* Red commands :red_circle:: main hotkeys needing CTRL
* Cyan commands: FPS limit hotkeys needing ALT

* Orange commands :orange_circle:: dig menu hotkeys
* Purple commands :purple_circle:: smoothing/engraving menu hotkeys
* Brown commands :brown_circle:: item property menu hotkeys

* Grey commands :black_circle:: advanced menu hotkeys (in dig/smooth/chop/gather menus)
